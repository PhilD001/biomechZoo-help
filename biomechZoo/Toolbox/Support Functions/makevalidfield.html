makevalidfield

PURPOSE

ch = makevalidfield(ch) fixes common invalid field names for structed

SYNOPSIS

function ch = makevalidfield(ch)

DESCRIPTION 

 ch = makevalidfield(ch) fixes common invalid field names for structed
 arrays
 
 ARGUMENTS

 RETURNS
   ch    ...   new name of channel as string with invalid fieldnames removed
