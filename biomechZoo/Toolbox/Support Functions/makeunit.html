makeunit

PURPOSE

MAKEUNIT makes all vectors unit vectors

SYNOPSIS

function r = makeunit(unt)

DESCRIPTION 

   MAKEUNIT  makes all vectors unit vectors
   unt ... N by 3 matrix of vectors.
           rows are the number of vectors
           columns are XYZ
