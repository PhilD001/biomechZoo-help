DESCRIPTION 

 TRAIN_TEST_SPLIT spliting ml_data into train test sets

 ARGUMENTS

   subjects     ...  string cell, subject name of all the subjects.

   split        ...  [0,1], percent of ml_data to be hold out for testing.
